# Spring Boot User API

REST API สำหรับระบบผู้ใช้ พร้อม JWT Authentication

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JWT
- MySQL
- Maven

## Features
- Register
- Login (JWT)
- Protected user APIs
- Validation
- Global Exception Handler
